Skills


When you have gotten a character card, the actions or the skills(except KKGs) which are on your card can be found here Skills can be aquired using xPs. The xPs of skills are stated at the end of the skill description. There are differnt classes of skills; melee skills, ranged skills, physical(unarmed and weapon skill)skills.


Types of Skills


There are six different types of skills, in the skill description, the types of skill it is is stated in a bracket after the skill name. These are the six skill types;


Trap Skill(trap): These are skills that activate when a certain condition is met either in opponents or users turn. Although the user can choose to activate it or not. They are sometimes called reation or bonus actions.


Attack Skill(atk): These type of skills mainly deal damage to opponents.


Time Skill(time): These are skills that take more than one turn to fufll their purpose.


Defensive Skill(def): These are skills that prevent you from losing health.


Neglecting Skill(neg): These skills stop effect of other skills based on the condition


Gain Skill(gain): These skills increase your stats, health, cp or even allow you play more than once.


Note


You can only use a particular skill once, unless it is stated in the description of the skill that it can be used more than once. But some skills take turns to recharge for use again thuo.A skill can be wasted, which means, if the move is used in a wrong situation, the move is counted but will have no effect in the battle.


A skill slot is a free zone in your card that skills can be kept. A skill slot can only hold one skill, but when the slot holds a mode, it can contain 3 modes.If a turn is skipped you lose 2hp but gain 2cp.


Players can sell skills for half the XPs it is worth. they are also skills that require certain weapons to be able to use them


If your hands are cut off you wont be able to use skills that requires hand, the admin will be the judge of such skills. Physical skills are skills like punch ,kick, non enchanted weapon skills and so on that don’t use cp.


You can only use 2 reations per turn.


Speed Rules


if you have 30 more speed than a target that target cant attack you with physical attack skills unless skills that cant be neglected

If you have 60 more speed than a target, they cant use defense skill against your attacks

If you have 100 more speed than a target, you can use an addition skill in your turns.

If you have 150 more speed than a target, your attacks cant be neglected by that target and direct attack skills from that target cant harm you.

Naturally in the real game , when you have more speed than other players, thier play time decreases
