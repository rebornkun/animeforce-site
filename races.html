Shinobi


shinobis race are similar to the human race only they are blessed with chakra with allows them to be able to control and use elemental abilities and do cities or country size damage. shinobi are basically strategic so to use one you have to be one.
Your elemental affinity tells you the type of elements you can use, certain clans can only use skills from the element(s):

  • fire release
  • water release
  • wind release
  • earth release
  • lightning release

but having more than one element and combining them can give you rise to others, a player that has more than one can use new elements if he has the right level and if its listed below;

  • wind + water +lv15 -- ice release
  • earth + water + lv10 -- Mud release
  • earth + water + lv15 -- wood release
  • fire + earth + lv15 -- lava release
  • lightning + water + lv20 -- storm release
  • water + fire + lv15 -- boil release
  • fire + earth + wind + lv25 -- dust release
  • earth + lightning + lv15 -- explosion release
  • fire + wind + lv15 -- scorch release
  • wind + earth + lv20 -- magnet release

you can gain elements even if not from your sub race, like from a tailed beast, a mode or weapon or others. those elements normally will only stay with you until that item or stuff giving it to you is no longer there. if you feel like they are other mixes that should be added message an admin.
Aside from the five elemental nature transformations, there are two nature transformations that are the source of all non-elemental techniques with are yin release and yang release, they give rise to techniques such as the Shadow Imitation Technique, Multi-Size Technique, medical ninjutsu, genjutsu, FuinJutsu
